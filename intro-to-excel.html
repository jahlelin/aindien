Intro To Excel

Jason Moore

These are my notes on an Intro To Excel.

Excel is the most used spreadsheet software in the world. It is part of the
Microsoft Office suite. Excel is used to perform numerical calculations. We use
Excel for number crunching, creating charts, organizing lists, text
manipulation, and accessing other data. 

Understanding Workbooks and Worksheets
Your work is done in a workbook. You can have many workbooks open at once. They
use an .xlsx extension. Workbooks will each open in a different window. 

Each workbook contains one or more worksheets. Each worksheet is made up of
individual cells. Each cell can contain a value, a formula, or text. A
worksheet can also contain charts, images, and diagrams. Worksheets are
available by clicking a tab at the bottom of the workbook. 

Every worksheet contains rows starting with the number 1. Columns start with the
letter "A". The intersection of a row and column is a single cell. Each cell has
a unique address made up of its column letter and row number. At any given time,
one cell is the active cell. It is the cell you have highlighted and it accepts
keyboard input. Its contents can be edited. You can identify the active cell by
its darker border. 

The Ribbon has various functions and tabs built in. We will cover them as they
come up. You can access the Ribbon by using keyboard commands just as easy as
the mouse. Use the "alt" key to display shortcuts on the Ribbon. Once you have
used the keys a few times you will start remembering some of them. This is how
you improve your workflow. 

Sometimes you will want to repeat a command. If you apply some particular
formatting to a cell then you can press ctrl-y to repeat it. 

There are also shorcut menus everywhere. You can right click just about anything
and get access to another menu. These menus are just another way to accomplish
certain tasks. 

Creating A Workbook
Start Excel with an empty workbook. If you have one open already you can create
a new one with the ctrl-n command. 

If you enter in a column of numbers work on as data, you can format them to make
them easier to read and understand. You can select all the cells in a range by
dragging with your mouse from beginning to the end. In the Number group on the
Ribbon, click the drop down Number Format and select currency or any other data
type you might be using. If currency, make sure and choose the two decimal place
too. 

We can now create a table with this data. Click on any cell that includes your
data. 

  •     Choose insert  
  •     Tables
  •     Table

 
Excel displays the create table box so youcan peruse the settings. Click ok and you get another tab to look at. You can choose different table types. 

Now, we can sum the data. 

  •     Activate any cell in the table
  •     Choose table tools
  •     Design
  •     Table Style Options
  •     Total Row

 

Excel automatically adds a new row to the bottom of your table. including a
formula that calculates the total of your data. If you would prefer to see a
different summary formula you can choose a different summary formula from the
drop down list. 

We can create a chart using a similar process. 

  •     Activate any cell in the table
  •     Choose insert
  •     Charts
  •     Recommended Charts
       

Select one of the options you see. You might even experiment and try a few
to see what you like for your data. Excel will insert the chart into the
center of your window. You can move it by dragging it in one direction. 

Printing Your Worksheet
Printing is easy if you have a printer available to you. 

  •     Click on any cell
  •     Click Page Layout to make sure everything looks good
  •     Choose File then Print

    Finally, click the Print button at the top

Saving Your Workbook
You will want to save your work so you do not lose it. 

  •     Click the Save button on the quick access bar
  •     It asks you to choose a location and name for your new file
  •     Click Browse to select the location
  •     In the File Name field, name it something that makes sense to you
  •     Click Save once more to have your file saved


By default, Excel saves a copy of your work every few min, but you should save
anyway to be sure you do not lose anything.
